# 💻 DevXirax Portfolio — Software Solutions Developer

Bienvenido a mi repositorio personal de presentación profesional. Aquí se encuentra el código de mi portfolio web, diseñado como una vitrina interactiva para mis capacidades en el desarrollo de **sistemas de gestión** y soluciones digitales a medida.

> **"Mi enfoque trasciende el código: es transformar procesos complejos en sistemas de gestión eficientes."**

---

## 🚀 Proyectos Destacados

Este portfolio centraliza tres de mis soluciones reales más impactantes:

1. **Bitácora de Vuelos (Aviación App)**:
   - **Ecosistema**: Mobile (Flutter & Firebase).
   - **Propósito**: Automatización de registros operativos y cálculo de autonomías legales para pilotos.
   - **Impacto**: App publicada en Play Store con más de 10k líneas de código y arquitectura por capas.

2. **Gestión Alfaverde**:
   - **Ecosistema**: Web (Python, Flask & local QR).
   - **Propósito**: Trazabilidad multimedia para pesajes de materiales en centros de reciclaje.
   - **Impacto**: Digitalización total del proceso de facturación y reducción de errores operativos.

3. **Gestión de Compras (Java UI)**:
   - **Ecosistema**: Desktop (Java 21, Swing & JSON).
   - **Propósito**: Administración de inventario y cierres financieros atómicos.
   - **Impacto**: Herramienta de alta velocidad diseñada para flujos industriales sin uso de ratón.

---

## 🛠️ Stack Tecnológico

Este sitio web fue construido utilizando **Frontend artesanal (Vanilla)** para garantizar el máximo rendimiento y control sobre el diseño:

- **HTML5 Semántico**: Optimizado para accesibilidad y SEO básico.
- **CSS3 Moderno**: Diseño responsivo "mobile-to-ultra-wide" con variables CSS, Flexbox y Grid.
- **JavaScript (ES6+)**: Sistema personalizado de modales dinámicos, **Lightbox (zoom de imágenes)** y animaciones de revelado por scroll.
- **Optimización**: Sin dependencias externas pesadas, asegurando una carga ultrarrápida.

---

## 📐 Características de Diseño

- **Arquitectura de Soluciones**: El diseño no solo lista proyectos, sino que explica el *Problema* vs. la *Solución* vs. el *Valor* generado.
- **Navegación Fluida**: Experiencia de usuario (UX) centrada en la claridad y la facilidad de contacto.
- **Showcase de Capturas**: Galería expandible para visualizar a detalle las interfaces de los sistemas reales.
